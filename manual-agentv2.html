1. 系统概述

民匠有约-服务商系统（代理商2.0）是面向代理商/服务商的专用管理平台，旨在帮助代理商高效管理企业客户、业务员团队、交易数据及激励奖励等核心业务。系统提供从企业信息维护、业务员管理到交易结算、激励奖励的全流程数字化管理能力。

系统特色

  • 多维度企业信息管理，支持代理区域与企业资质维护
  • 灵活的业务员（个人代理）管理，支持邀请码体系
  • 完整的企业客户列表，覆盖拉新方与落地服务方双维度
  • 购买明细与结算明细的精准追踪与导出
  • 激励奖励体系，清晰展示奖励来源与结算状态
  • 便捷的数据筛选与Excel导出功能

适用对象

角色 说明
代理商管理员 管理所属代理商下的全部业务与数据
业务员 通过邀请码拓展企业客户，查看个人业务数据

2. 登录系统

代理商2.0系统支持多种登录方式，确保账号安全与操作便捷。

2.1 访问地址

打开浏览器，访问系统地址：http://agentv2.zjmjyy.com/

提示

建议使用 Chrome、Edge 等主流浏览器，分辨率建议 1920×1080 及以上。

2.2 登录方式

系统支持以下两种登录方式：

方式一：手机号 + 密码 + 短信验证码登录

1在登录页面输入已注册的手机号

2输入对应的登录密码

3点击「获取验证码」按钮，系统将向该手机号发送短信验证码

4输入收到的短信验证码

5点击「登录」按钮进入系统

方式二：账号 + 密码 + 短信验证码登录

1在登录页面输入账号（非手机号时使用）

2输入对应的登录密码

3点击「获取验证码」，输入收到的短信验证码

4点击「登录」按钮进入系统

2.3 注册账号

1在登录页面点击「注册账号」链接

2填写手机号、密码等注册信息

3获取并输入短信验证码

4点击「注册」完成账号创建

2.4 忘记密码

1在登录页面点击「忘记密码」链接

2输入注册时使用的手机号

3获取短信验证码并验证身份

4设置新密码并确认
注意

短信验证码有效时间有限，请及时输入。如未收到验证码，请检查手机短信拦截设置或稍后重试。

3. 企业信息

企业信息模块用于维护代理商所属企业的基本信息和资质资料，是系统核心基础数据。

3.1 信息字段说明

字段名称 说明 是否必填
代理区域 代理商所负责的业务区域范围 是
企业Id 系统自动分配的企业唯一标识 系统生成
邀请码 用于邀请企业入驻的唯一邀请码 系统生成
组织类型 企业的组织类型分类 是
营业执照扫描件 企业营业执照的扫描件/照片上传 是
企业合同 与合作方签署的企业合同文件 是

3.2 法定代表人信息

字段名称 说明 是否必填
法定代表人姓名 企业法定代表人的真实姓名 是
身份证号码 法定代表人的身份证号码 是
手机号 法定代表人的联系电话 是

3.3 操作步骤

1登录系统后，在左侧菜单点击「企业信息」

2查看当前企业的基本信息，包括代理区域、企业Id、邀请码等

3如需修改，点击对应字段的「编辑」按钮进行更新

4上传营业执照扫描件和企业合同文件（支持常见图片及PDF格式）

5填写或更新法定代表人信息

6点击「保存」提交修改
提示

企业Id和邀请码由系统自动生成，不可手动修改。邀请码可用于邀请企业入驻，请妥善保管。

注意

营业执照扫描件需清晰完整，确保信息可辨识。企业合同需上传已签署的正式合同文件。

4. 个人代理

个人代理模块用于管理代理商下的业务员团队，支持业务员的新增、查看和搜索。

4.1 业务员列表

业务员列表展示当前代理商下所有业务员的详细信息，字段说明如下：

字段名称 说明
邀请码 业务员的唯一邀请码，用于关联企业客户
姓名 业务员真实姓名
身份证号码 业务员身份证号码
手机号 业务员联系电话
归属企业 业务员所属的企业名称
类别 业务员的分类/级别

4.2 新增业务员

1在「个人代理」页面，点击「新增业务员」按钮

2在弹出的表单中填写业务员信息：
  • 姓名（必填）
  • 身份证号码（必填）
  • 手机号（必填）
  • 归属企业（必填）
  • 类别（必填）
3确认信息无误后，点击「确定」提交

4系统自动生成该业务员的邀请码

4.3 搜索业务员

1在业务员列表上方的搜索区域，选择搜索条件

2按业务员搜索：输入业务员姓名进行模糊查询

3按手机号搜索：输入手机号进行精确查询

4点击「搜索」按钮，列表将显示匹配结果
提示

新增业务员后，系统会自动分配邀请码。该邀请码与业务员一一对应，用于追踪其拉新的企业客户。业务员姓名支持模糊搜索，手机号为精确匹配。

5. 企业列表

企业列表模块展示代理商所关联的全部企业客户信息，包含拉新方和落地服务方两个维度的数据。

5.1 列表字段说明

字段名称 说明
注册地 企业的注册所在地
拉新方邀请码 拉新方业务员的邀请码
拉新方 负责拉新该企业的业务员/代理商名称
企业名称 企业全称
工商识别号 企业统一社会信用代码
法人姓名 企业法定代表人姓名
注册手机号 企业注册时使用的手机号
落地服务邀请码 落地服务方业务员的邀请码
落地服务方 提供落地服务的业务员/代理商名称
如意宝（席位） 该企业购买的如意宝产品席位数
福运宝（结算） 该企业购买的福运宝产品结算数
安心云（席位） 该企业购买的安心云产品席位数
激励奖励金额 该企业产生的激励奖励金额

5.2 激励奖励金额总计

企业列表页面底部展示激励奖励金额总计，汇总当前列表所有企业的激励奖励金额之和，方便代理商快速了解整体收益情况。

5.3 操作步骤

1在左侧菜单点击「企业列表」

2浏览企业客户列表，查看各企业的详细信息

3关注页面底部的激励奖励金额总计

4如需查找特定企业，可使用页面搜索功能按企业名称等条件筛选
提示

拉新方与落地服务方可以不同，一个企业可能由不同业务员分别负责拉新和落地服务。如意宝、福运宝、安心云的数值反映企业实际购买使用情况，与激励奖励金额挂钩。

6. 购买明细

购买明细模块记录所有企业客户的产品购买记录，支持多条件筛选和数据导出。

6.1 列表字段说明

字段名称 说明
购买时间 产品购买的具体时间
拉新方 负责拉新的业务员/代理商名称
企业名称 购买产品的企业全称
落地服务 提供落地服务的业务员/代理商名称
产品 购买的产品名称（如如意宝、福运宝、安心云等）
价格 产品单价
数量 购买数量
付款金额 实际付款金额（价格 × 数量）

6.2 统计信息

购买明细页面展示以下统计数据：

  • 交易成功总金额：所有交易成功订单的付款金额总和
  • 交易成功总笔数：所有交易成功订单的数量

6.3 筛选条件

支持以下筛选条件组合查询：

筛选条件 说明
购买时间 选择时间范围，支持起止日期
拉新方 按拉新方名称筛选
企业名称 按企业名称筛选
落地服务 按落地服务方筛选
产品类型 按产品分类筛选（如意宝/福运宝/安心云等）

6.4 导出Excel

1设置好筛选条件（可选，不设则导出全部数据）

2点击页面上的「导出Excel」按钮

3系统将根据当前筛选条件生成Excel文件并下载

6.5 操作步骤

1在左侧菜单点击「购买明细」

2查看购买记录列表及统计信息

3如需筛选，在页面顶部设置筛选条件后点击「查询」

4如需导出，点击「导出Excel」下载数据文件
提示

导出的Excel数据将反映当前筛选条件下的结果，建议先确认筛选条件再导出，以确保数据准确性。

7. 结算明细

结算明细模块记录所有任务结算的详细信息，包含税费计算和到账金额，支持多条件筛选和数据导出。

7.1 列表字段说明

字段名称 说明
拉新方 负责拉新的业务员/代理商名称
企业名称 任务所属企业全称
落地服务 提供落地服务的业务员/代理商名称
任务编号 系统分配的任务唯一编号
姓名 任务执行人员的姓名
身份证号 任务执行人员的身份证号码
付款金额 任务结算的付款金额
税率 适用的税率百分比

7.3 筛选条件

筛选条件 说明
拉新方 按拉新方名称筛选
企业名称 按企业名称筛选
落地服务 按落地服务方筛选
姓名 按任务执行人员姓名筛选
时间选择 按任务完成时间范围筛选

7.4 导出Excel

1设置好筛选条件（可选，不设则导出全部数据）

2点击页面上的「导出Excel」按钮

3系统将根据当前筛选条件生成Excel文件并下载

7.5 操作步骤

1在左侧菜单点击「结算明细」

2查看结算记录列表及交易结算总金额

3如需筛选，在页面顶部设置筛选条件后点击「查询」

4如需导出，点击「导出Excel」下载数据文件

到账金额 = 付款金额 - 税费。税费由系统根据付款金额和适用税率自动计算，无需手动输入。

9. 个人中心

个人中心提供账户安全相关的管理功能，通过右上角用户名下拉菜单访问。

9.1 修改密码

1点击页面右上角的用户名，展开下拉菜单

2选择「修改密码」选项

3在弹出的修改密码对话框中：
  • 输入旧密码（当前使用的密码）
  • 输入新密码（需符合密码强度要求）
  • 输入确认密码（与新密码一致）
4点击「确定」按钮完成密码修改

5密码修改成功后需重新登录

9.2 退出登录

1点击页面右上角的用户名，展开下拉菜单

2选择「退出登录」选项

3系统将退出当前账号，返回登录页面
注意

修改密码后系统将自动退出登录，请使用新密码重新登录。建议定期修改密码，确保账号安全。请勿将账号密码告知他人，避免账号被盗用。

附录：常见问题

Q: 忘记密码怎么办？
A: 在登录页面点击「忘记密码」，通过手机号验证后重置密码。

Q: 短信验证码收不到怎么办？
A: 请检查手机短信拦截设置，确认手机信号正常。如仍无法收到，请稍后重试或联系平台客服。

Q: 导出的Excel数据为空怎么办？
A: 请检查筛选条件是否过于严格导致无匹配数据。尝试放宽筛选条件或清除筛选后重新导出。

Q: 业务员的邀请码在哪里查看？
A: 在「个人代理」模块的业务员列表中，每条业务员记录均展示其对应的邀请码。

Q: 激励奖励金额与企业列表中的金额不一致？
A: 企业列表中的激励奖励金额为该企业的累计总额，激励奖励模块展示的是每笔奖励的明细记录，二者为汇总与明细的关系。

Q: 页面显示不全怎么办？
A: 建议使用 Chrome 或 Edge 浏览器，分辨率建议 1920×1080 以上。
