1. 产品概述

1.1 产品背景

民匠有约管理系统是面向企业的综合型人力资源与任务管理平台，旨在通过数字化手段提升企业管理效率，降低运营成本，为企业提供一站式的人力、培训、财务、任务管理解决方案。

1.2 产品目标

  • 实现用户全生命周期管理
  • 提供在线培训与考试能力
  • 建立灵活的薪资结算体系
  • 提供多维度数据统计分析
  • 保障系统安全与数据隐私

1.3 目标用户

  • 系统管理员：负责系统配置与维护
  • 企业管理员：负责企业内部人员与任务管理
  • 财务人员：负责财务对账与结算
  • 培训管理员：负责培训课程与学员管理
  • 普通用户：使用平台提供的各项服务

1.4 产品定位

一站式人力资源与任务管理 SaaS 平台，服务于中大型企业的数字化转型需求。

2. 用户角色与权限

2.1 角色定义

角色描述主要权限
超级管理员系统最高权限全部功能权限、用户管理、系统配置
系统管理员系统运维管理系统管理模块、日志查看、监控
企业管理员企业运营管理用户管理、任务管理、薪资审核
财务人员财务相关操作财务管理、统计报表、对账开票
培训管理员培训相关管理培训班级、学员管理、习题管理
运营人员运营推广管理商户管理、服务商管理、消息通知
普通用户基础功能使用个人信息、任务查看、学习记录

2.2 权限设计原则

  • 最小权限原则：用户仅拥有完成工作所需的最小权限
  • 角色继承：高级角色默认拥有低级角色权限
  • 数据隔离：不同企业间数据完全隔离
  • 操作审计：所有关键操作记录审计日志

4. 详细功能需求

4.1 登录系统

功能描述：用户通过账号密码验证码登录系统。

输入项：

  • 账号：文本输入，必填
  • 密码：密码输入，必填，支持显示/隐藏
  • 验证码：图形验证码，必填

处理逻辑：

  1. 验证账号密码是否正确
  2. 验证验证码是否匹配
  3. 验证通过后生成登录 Token
  4. 记录登录日志
  5. 跳转到首页

输出项：

  • 登录成功：跳转首页
  • 登录失败：显示错误提示

业务规则：

  • 连续 5 次登录失败，账号锁定 30 分钟
  • 密码长度不少于 8 位，包含字母和数字
  • 验证码有效期 5 分钟

4.2 首页仪表盘

功能描述：展示系统核心数据概览。

页面元素

  • 用户统计：总用户数、新增用户、活跃用户
  • 完成率统计：任务完成率、培训完成率

2. 数据图表（4个）

  • 培训趋势图：折线图，支持天/月切换
  • 任务趋势图：折线图，支持天/月切换
  • 用户增长趋势：折线图，支持天/月切换
  • 完成率对比：柱状图

交互要求：

  • 点击天/月按钮切换图表时间维度
  • 切换时有加载动画和成功提示
  • 鼠标悬停显示详细数据

5. 非功能需求

5.1 性能需求

指标要求
页面加载时间首屏 < 3s，其他页面 < 2s
接口响应时间普通查询 < 500ms，复杂查询 < 2s
并发用户数支持 1000+ 并发用户
系统可用性99.9% 以上

5.2 安全需求

  • 用户密码加密存储，使用 BCrypt 算法
  • 敏感数据传输使用 HTTPS
  • SQL 注入、XSS、CSRF 防护
  • 接口权限校验，防止越权访问
  • 操作日志完整记录，支持审计
  • 定期自动备份数据

5.3 兼容性需求

  • 浏览器：Chrome 90+、Edge 90+、Firefox 88+、Safari 14+
  • 分辨率：最小支持 1366*768，推荐 1920*1080
  • 移动端：响应式布局，支持平板访问

5.4 可用性需求

  • 操作反馈：所有操作有明确的成功/失败提示
  • 加载状态：数据加载显示 Loading 动画
  • 错误提示：友好的错误信息，避免技术术语
  • 帮助文档：关键功能提供操作指引

  • 圆角：4px 小圆角

6.2 布局规范

  • 侧边栏宽度：210px，折叠后 64px
  • 顶部导航高度：60px
  • 内容区左右内边距：20px
  • 卡片间距：20px

6.3 组件规范

  • 使用 Element Plus 组件库
  • 表格必须有边框和斑马纹
  • 弹窗宽度：小型 400px、中型 600px、大型 800px
  • 按钮主操作使用 type="primary"
  • 危险操作使用 type="danger" + 确认弹窗
