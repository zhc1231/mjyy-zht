1. 系统概述

民匠有约-企业后台是面向企业管理员的专用操作平台，旨在帮助企业高效管理企业信息、任务发布、人员管理、结算对账等核心业务。系统提供从企业入驻、认证审核到任务发布、人员管理、薪资结算的全流程数字化管理能力。

1.1 系统特色

  • 一站式任务管理平台，支持从工种配置到薪资结算的全流程管理
  • 完善的企业认证流程，实时显示认证状态
  • 灵活的任务发布功能，支持多工种、多人数任务配置
  • 清晰的结算记录，服务费、税费明细一目了然
  • 考勤打卡数据自动同步，工时统计准确高效
  • 账户充值与余额实时管理，资金流向清晰透明
  • 便捷的数据筛选与搜索功能
  • 响应式布局，适配不同分辨率

1.2 适用对象

角色说明
企业管理员管理所属企业的全部业务与数据，包括任务发布、人员管理、结算对账等
企业法人查看企业认证状态、项目信息、财务数据等

1.3 功能模块总览

模块名称功能说明
首页概览核心数据统计展示，任务、席位、交易流水概览
任务指南任务操作流程指引，含工种配置、排期、人才库等
任务中心任务发布、管理、状态跟踪
结算记录薪资结算明细，服务费、税费计算
打卡机数据考勤打卡数据统计与查看
账户管理企业账户充值、余额管理
项目管理企业项目管理，任务按项目维度分组
系统管理系统基础配置与企业信息维护

2. 登录系统

企业后台支持手机号/账号登录方式，确保账号安全与操作便捷。

2.1 访问地址

打开浏览器，访问企业后台地址：http://121.199.7.40:9093

建议使用 Chrome、Edge 等主流浏览器，分辨率建议 1920×1080 及以上。

2.2 登录方式

方式一：手机号 + 密码 + 短信验证码登录

  1. 在登录页面输入已注册的手机号
  2. 输入对应的登录密码
  3. 点击「获取验证码」按钮，系统将向该手机号发送短信验证码
  4. 输入收到的短信验证码
  5. 点击「登录」按钮进入系统

方式二：账号 + 密码 + 短信验证码登录

  1. 在登录页面输入账号（非手机号时使用）
  2. 输入对应的登录密码
  3. 点击「获取验证码」，输入收到的短信验证码
  4. 点击「登录」按钮进入系统

2.3 注册账号

  1. 在登录页面点击「注册账号」链接
  2. 填写手机号、密码等注册信息
  3. 获取并输入短信验证码
  4. 点击「注册」完成账号创建

2.4 忘记密码

  1. 在登录页面点击「忘记密码」链接
  2. 输入注册时使用的手机号
  3. 获取短信验证码并验证身份
  4. 设置新密码并确认
短信验证码有效时间有限，请及时输入。如未收到验证码，请检查手机短信拦截设置或稍后重试。

3. 企业管理

企业管理模块用于管理用户关联的所有企业信息，包括企业列表查看、认证状态管理、新增企业等功能。

3.1 企业列表

企业列表页面展示当前用户关联的所有企业信息，包括企业名称、创建时间、认证状态等。

列表字段说明

字段名称说明
企业名称企业全称
创建时间企业入驻系统的时间
认证状态企业认证状态（认证通过/审核中/认证失败）

操作步骤

  1. 登录系统后，自动进入企业列表页面
  2. 浏览企业列表，查看各企业的详细信息
  3. 如需进入项目管理，点击企业对应的「进入项目」按钮
  4. 如需创建新企业，点击页面右上角「新增企业」按钮

3.2 企业认证

企业认证是企业入驻系统的必要步骤，认证通过后企业才能正常使用系统功能。

认证状态说明

状态说明
认证通过企业资质已审核通过，可正常使用系统
审核中企业资质正在审核中，请耐心等待
认证失败企业资质审核未通过，需重新提交资质材料
认证失败时，系统会提示失败原因，请根据提示修正后重新提交。

3.3 新增企业

企业管理员可以创建新的企业信息，填写企业基本信息并上传资质材料。

操作步骤

  1. 在企业列表页面，点击右上角「新增企业」按钮
  2. 填写企业基本信息：企业名称、统一社会信用代码、法人信息等
  3. 上传企业资质：营业执照扫描件、企业合同等
  4. 确认信息无误后，点击「提交」
  5. 等待系统审核企业资质

必填信息

字段名称说明是否必填
企业名称企业全称是
统一社会信用代码企业工商识别号是
法人姓名企业法定代表人姓名是
法人身份证号企业法定代表人身份证号码是
营业执照扫描件企业营业执照的扫描件/照片是
企业合同与合作方签署的企业合同文件是

4. 首页概览

首页概览是企业后台的数据仪表盘，集中展示企业核心经营数据和快捷操作入口，帮助企业管理员快速了解企业运营状况。

4.1 数据概览

账户资金概览

指标名称说明
累计充值金额企业账户历史累计充值的总金额
当前剩余金额企业账户当前可用的余额
待发放金额已计入但尚未发放的薪资金额
已发放金额历史累计已发放的薪资总金额

任务管理统计

指标名称说明
进行中任务数当前正在进行中的任务数量
总发布数历史累计发布的任务总数
在进行中的任务人次数当前正在执行任务的人员数量
团队总人数企业团队的总人数

席位管理统计

指标名称说明
已购席位数企业已购买的席位总数
团队总人数企业团队的总人数
今日使用数今日实际使用的席位数量

4.2 交易流水

交易流水模块展示企业的资金流动情况，支持按时间维度筛选查看。

操作步骤

  1. 在首页找到「交易流水」区域
  2. 点击切换时间维度：今日 / 昨日 / 上月 / 本月
  3. 查看对应时间段的发放金额、服务费、税费数据
  4. 点击「查看更多」按钮，进入详细的交易流水页面

4.3 发放记录排行

发放记录排行展示薪资发放的人员排名，便于了解人员薪资分布情况。

字段名称说明
排名按结算金额从高到低的排名
姓名人员姓名
结算金额该人员的薪资结算金额
服务费该笔结算产生的服务费
税费该笔结算产生的税费

4.4 任务列表概览

首页展示最新的任务列表，便于快速了解任务进展情况。

字段名称说明
任务任务名称/描述
状态任务当前状态（进行中、已完成、已取消等）
任务周期任务的开始日期和结束日期
人数参与该任务的人员数量
预计结算任务预计的薪资结算金额
实际结算任务实际的薪资结算金额

4.5 操作流程指南

首页顶部展示完整的任务操作流程，引导用户按步骤完成任务管理：

  1. 工种配置：配置任务所需的工种类型
  2. 任务排期：规划任务时间安排
  3. 人才库：管理企业人才资源
  4. 发布任务：创建并发布新任务
  5. 任务详情：查看任务详细信息
  6. 添加成员：为任务添加执行人员
  7. 完成：任务完成与结算
点击首页的「发布任务」按钮，可快速进入任务发布页面，提高操作效率。

5. 任务指南

任务指南模块为企业管理员提供任务操作的完整流程指引，帮助快速上手任务管理功能。

5.1 任务工种

任务工种配置是任务管理的第一步，用于定义任务所需的各工种类型。

操作说明

  • 在左侧菜单点击「任务指南」→「1.任务工种」
  • 查看和管理企业支持的工种类型
  • 根据实际业务需求配置所需工种

5.2 任务排期

任务排期用于规划任务的时间安排，确保任务按时执行。

操作说明

  • 在左侧菜单点击「任务指南」→「2.任务排期」
  • 设置任务的开始时间和结束时间
  • 合理安排任务周期，避免时间冲突

5.3 人才库

人才库是企业的人才资源池，用于管理和调度企业可用的人力资源。

操作说明

  • 在左侧菜单点击「任务指南」→「3.人才库」
  • 查看企业人才库中的人员信息
  • 根据工种、技能等条件筛选合适的人员

5.4 任务列表

任务列表是任务管理的核心操作页面，用于创建、查看和管理任务。

操作说明

  • 在左侧菜单点击「任务指南」→「4.任务列表」
  • 查看所有任务的列表和状态
  • 创建新任务、查看任务详情、管理任务成员

6. 任务中心

任务中心是企业任务的统一管理平台，支持任务发布、状态跟踪、人员管理等全流程操作。

6.1 发布任务

企业管理员可以发布新任务，配置任务的工种、人数、时间等信息。

操作步骤

  1. 在左侧菜单点击「任务中心」
  2. 点击「发布任务」按钮
  3. 填写任务基本信息：任务名称、任务描述等
  4. 选择任务所需的工种和各工种人数
  5. 设置任务周期（开始日期、结束日期）
  6. 确认信息无误后，点击「发布」
  7. 任务发布成功，可在任务列表中查看

6.2 任务状态管理

任务在不同阶段有不同的状态，便于跟踪任务进展。

状态说明
进行中任务正在执行过程中
已完成任务已完成并结算
已取消任务被取消，不再执行

6.3 任务人员管理

任务发布后，可以为任务添加执行人员，管理人员配置。

操作说明

  • 在任务列表中点击目标任务，进入任务详情
  • 查看当前任务的成员列表
  • 添加或移除任务成员
  • 调整各工种的人员配置
任务人员数量应与任务发布时的工人数相匹配，确保任务正常执行。

7. 结算记录

结算记录模块用于管理和查看任务薪资结算明细，包括发放金额、服务费、税费等详细信息。

7.1 结算明细查看

操作步骤

  1. 在左侧菜单点击「结算记录」
  2. 查看所有结算记录列表
  3. 可按时间、任务、人员等条件筛选结算记录
  4. 点击具体记录查看结算详情

7.2 费用明细说明

费用项目说明
发放金额实际发放给人员的薪资金额
服务费平台收取的服务费用
税费按照国家规定收取的税费
预计结算金额与实际结算金额可能存在差异，以实际结算金额为准。

8. 打卡机数据

打卡机数据模块用于查看和统计人员的考勤打卡数据，为工时计算和薪资结算提供依据。

8.1 打卡数据查看

操作步骤

  1. 在左侧菜单点击「打卡机数据」
  2. 查看人员的打卡记录
  3. 可按日期、人员、工种等条件筛选
  4. 导出打卡数据用于统计分析

8.2 工时统计

系统基于打卡数据自动计算人员工时，确保工时统计准确高效。

  • 上班打卡时间与下班打卡时间自动计算工时
  • 支持加班工时的识别与统计
  • 工时数据直接关联薪资结算

9. 账户管理

账户管理模块用于管理企业的充值账户，查看余额、充值记录等资金相关信息。

9.1 账户充值

操作步骤

  1. 在左侧菜单点击「账户管理」
  2. 点击「充值」按钮
  3. 输入充值金额
  4. 选择支付方式
  5. 完成支付，充值金额实时到账

9.2 余额查询

企业账户余额实时展示，可在首页和账户管理页面查看。

  • 当前剩余金额：账户可用余额
  • 累计充值金额：历史充值总额
  • 已发放金额：历史发放总额

9.3 充值记录

支持查看历史充值记录，包括充值时间、充值金额、支付方式等信息。

10. 项目管理

项目管理模块用于管理企业的所有项目，支持任务按项目维度进行分组管理。

10.1 项目列表

项目列表页面展示当前企业下的所有项目信息，项目列表标题会显示企业名称前缀。

操作步骤

  1. 在左侧菜单点击「项目管理」
  2. 进入项目列表页面，查看当前企业下的所有项目
  3. 如需创建新项目，点击页面右上角「新增项目」按钮

10.2 新增项目

企业管理员可以为企业创建新项目，便于按项目维度管理任务。

操作步骤

  1. 在项目列表页面，点击右上角「新增项目」按钮
  2. 填写项目基本信息：项目名称、项目描述等
  3. 确认信息无误后，点击「提交」
  4. 项目创建成功，自动跳转到项目列表

必填信息

字段名称说明是否必填
项目名称项目全称是
项目描述项目简要说明否

10.3 项目与任务关联

发布任务时可以关联项目，便于按项目维度查看和统计任务数据。

  • 任务发布时选择所属项目
  • 在项目详情中查看该项目下的所有任务
  • 按项目维度统计任务完成情况和结算数据
项目名称应清晰明确，便于后续项目管理和查找。

11. 系统管理

系统管理模块用于企业的基础配置和信息维护，保障系统正常运行。

11.1 企业信息管理

企业管理员可以维护企业的基本信息，确保企业资料准确完整。

操作步骤

  1. 在左侧菜单点击「系统管理」
  2. 进入企业信息管理页面
  3. 查看和编辑企业基本信息
  4. 修改完成后点击「保存」

11.2 系统设置

系统设置提供企业的个性化配置选项。

  • 企业基本资料维护
  • 联系人信息管理
  • 通知设置等个性化配置
修改企业信息后，如需更新认证信息，请联系平台客服进行审核。
